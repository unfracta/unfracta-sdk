# Security Policy

If you discover a security issue, please report it privately using GitHub
Security Advisories for this repository. We will acknowledge receipt and
provide a remediation plan or timeline as soon as possible.

## Supported Versions

Security fixes are applied to the latest tagged release on the `main` branch.
Older tags are not guaranteed to receive backports.
